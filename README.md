# Kofi K. Profile

This is a profile page for Kofi K. It is built with React and Vite.

Visit the main site at [kofi-k](https://kofik.github.io/kofi-k-profile-page/)


<hr/>
## Installation with npm or yarn

```bash
yarn install
```

if you use npm

```bash
npm install
```

## Run the development server

```bash
yarn dev
```

for npm

```bash
npm run dev
```
